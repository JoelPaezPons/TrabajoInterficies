Proyecto de Réplica Web - Interfaces 🎨
Este repositorio contiene un proyecto práctico de diseño de interfaces web donde el objetivo principal fue replicar con la máxima fidelidad posible un sitio web real o un diseño específico utilizando estándares modernos de desarrollo frontend.

🎯 Objetivo del Proyecto
El desafío consistió en analizar visualmente una web existente y reconstruirla desde cero, prestando especial atención a:

Fidelidad Visual: Mantener proporciones, espaciados y jerarquía visual.

Componentización: Uso eficiente de elementos de interfaz.

Adaptabilidad: Asegurar que el diseño sea totalmente responsivo.

🛠️ Tecnologías y Herramientas
HTML5: Estructura semántica clara.

CSS3: Estilos personalizados, Flexbox y Grid para layouts complejos.

Bootstrap 5: Implementación del sistema de rejilla (grid), botones, navegación y utilidades para acelerar el desarrollo.

Assets: Gestión de imágenes y fuentes externas.

📂 Estructura del Proyecto
index.html: Estructura principal de la réplica.

styles.css: Estilos personalizados para ajustar detalles que Bootstrap no cubre por defecto.

img/: Directorio con las imágenes y recursos visuales utilizados.

Proyectos separados/: Diferentes versiones o secciones específicas del desarrollo.
